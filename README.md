# AOTC
Автоматизация ведения бухгалтерии для ИП на УСН "Доходы" в MS Excel в 2026 году. 

Файлы предназначены для использования селлерами, работающими только с маркетплейсом Wildberries. 

Функционал: расчет ежеквартальных сумм авансов налога по УСН и сумм авансов НДС по данным отчетов о реализации товара на маркетплейсе Wildberries.

Состав файлов:
1) WB_nalog_USN_NDS.xlsm - если вы доверяете макросам из внешних источников - используйте только этот, готовый к работе файл с макросом.
2) WB_nalog_USN_NDS.xlsx - если не доверяете макросам из внешних источников - используйте этот файл (шаблон без макроса) и файл с макросом.
3) Module1.bas - макрос, который будет нужно импортировать в WB_nalog_USN_NDS.xlsx
4) WB_nalog_USN_NDS_short.docx - краткое руководство пользователя
   
КАК ИМПОРТИРОВАТЬ:
1. Откройте WB_nalog_USN_NDS.xlsx в Excel
2. Alt+F11 > Редактор VBA
3. Файл > Импорт файла > выберите Module1.bas
4. Удалите, если есть, старый модуль (ПКМ > Remove Module)
5. Закройте редактор, сохраните файл как .xlsm
