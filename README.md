# AOTC
Автоматизация ведения бухгалтерии для ИП на УСН "Доходы". 
Предназначена для селлеров, работающих только с маркетплейсом Wildberries. 
Расчет налога по УСН и НДС на данных отчетов маркетплейса.

Состав файлов:
1) WB_nalog_USN_NDS.xlsm - используйте готовый к работе файл с макросом.
2) WB_nalog_USN_NDS.xlsx - шаблон без макроса. используйте в том случае, если как и я не доверяете макросам из внешних источников
3) Module1.bas - макрос, который будет нужно импортировать в WB_nalog_USN_NDS.xlsx
4) WB_nalog_USN_NDS_short.docx - краткое руководство пользователя
   
КАК ИМПОРТИРОВАТЬ:
1. Откройте WB_nalog_USN_NDS.xlsx в Excel
2. Alt+F11 > Редактор VBA
3. Файл > Импорт файла > выберите Module1.bas
4. Удалите, если есть, старый модуль (ПКМ > Remove Module)
5. Закройте редактор, сохраните файл как .xlsm
